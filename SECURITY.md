# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |

## Reporting a Vulnerability

Report a vulnerability by going to the [Issues](https://github.com/colack/mdstfu/issues) tab and submitting a security report.
